# MediNotes 5 分钟快速入门

## 🚀 三步开始使用

### Step 1: 选择功能（10 秒）

```
┌─────────────────────────────────────┐
│  http://localhost:3000              │
├─────────────────────────────────────┤
│                                     │
│  🔍 Research  - 医学知识查询         │
│  ⚠️  Verify   - 药物交互检查         │
│  📄 Document  - 会诊摘要生成         │
│  📚 History   - 历史记录查看         │
│                                     │
└─────────────────────────────────────┘
```

---

### Step 2: 输入查询（30 秒）

#### Research 示例
```
What are the latest treatment guidelines for Type 2 Diabetes?
```

#### Verify 示例
```
Warfarin
Aspirin
Lisinopril
```

#### Document 示例
```
65 yo male with DM2, presenting with SOB x 3 days...
```

---

### Step 3: 查看结果（3 分钟）

✅ **Research**: 阅读 AI 总结 + 引用来源  
✅ **Verify**: 查看交互报告 + FDA 链接  
✅ **Document**: 审阅生成摘要 + 编辑完善  

---

## 💡 快速技巧

### Research 技巧
- ✅ 先问大方向（指南、概述）
- ✅ 再问具体细节（副作用、用量）
- ✅ 点击引用查看原文

### Verify 技巧
- ✅ 每行输入一个药物
- ✅ 使用通用名（不是商品名）
- ✅ 点击 FDA 链接查看详情

### Document 技巧
- ✅ 提供详细信息
- ✅ 分段输入更清晰
- ✅ **人工审核很重要**

## ⚠️ 重要提醒

## 📱 快速演示

### 演示 1: 查询药物副作用（1 分钟）

```
1. 点击 Research
2. 输入: "Metformin side effects"
3. 点击搜索
4. 阅读结果 ✅
```

---

### 演示 2: 检查药物交互（1 分钟）

```
1. 点击 Verify
2. 输入药物：
   Warfarin
   Aspirin
3. 点击「開始分析」
4. 查看交互报告 ✅
```

---

### 演示 3: 生成会诊摘要（3 分钟）

```
1. 点击 Document
2. 粘贴患者信息
3. 点击「Generate Consultation Summary」
4. 查看生成的摘要 ✅
5. 编辑和完善 ✅
```

---

## 🆘 遇到问题？

### 常见问题

**Q: 查询很慢？**
- A: 首次查询需要 2-3 秒，再次查询会更快（缓存）

**Q: 找不到某个药物？**
- A: 可能药物名称拼写错误，或 FDA 数据库中没有

**Q: 生成的摘要不准确？**
- A: 请提供更详细的信息，并人工审核修改
